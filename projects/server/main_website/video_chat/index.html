Plug in not installed

It seems that the webrtc plugin did not load. Have you installed the plugin already?

Mac: To install the plugin you need to:

  1. Download WebrtcPlugin.plugin.
  2. Unzip the file. Copy WebrtcPlugin.plugin into "~/Library/Internet Plug-Ins".

Linux: To install the plugin you need to:

  1. Download npWebrtcPlugin.so.
  2. Untar the file. Copy npWebrtcPlugin.so into "~/.mozilla/plugins".
