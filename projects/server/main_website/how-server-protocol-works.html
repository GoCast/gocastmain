GoCast

The functions specified below describe the basic set of APIs provided by the plugin at this time. We have made a conscious effort to keep it simple for now, so that developers can quickly embed communications into their web applications. In the future, we will make the APIs reflect more and more of those specified in W3C's WebRTC API spec.

Method to initiate sign-in to the specified signin server. Signin(string: peerName, string: serverIp, int: serverPort)

Method to sign out from the signin server. Signout()

Method to initiate an audio peer connection with desired peer. Call(string: remotePeerName)

Method to initiate a disconnection with desired peer. Hangup(string: remotePeerName)

Events fired by the plugin which are to be handled in JavaScript:
1. eventType: 'SignedIn', argument: list of online peers (grammar: [:]*).
2. eventType: 'PeerOnline', argument: name of newly online peer.
3. eventType: 'PeerOffline', argument: name of newly offline peer.
4. eventType: 'RemotePeerCall', argument: name of remote peer who just initiated an audio connection.
5. eventType: 'RemotePeerHangup', argument: name of remote peer who just disconnected.
