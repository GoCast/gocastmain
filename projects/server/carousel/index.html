In order to participate in a meeting you will need the Gocast.it plugin installed.

Please click here Download Gocast.it.plugin to download. Then click on the downloaded file to install.

Then, after downloading the .tar.gz file, follow these steps to install:

  1. Extract the files with tar xzvf GoCastPlayer_XXX.tar.gz
  2. Go to the directory created by cd GoCastPlayer
  3. Execute the install script ./install.sh
  4. Restart your browser.

... and You are ready to have fun with the GoCast Player.
